黑苹果M.2接口SSD由外置改为内置

2019-11-15 /
  • 折腾
  • 黑苹果
 / 已读:0

解决方案

有2个，一个是改DSDT，今天是另外一个，其实我搜了很久，都是建议改DSDT的，心累，最后在百度帖吧看到了一个装LILU插件的，可是却并没有附件(搬帖不全)

知道插件名字叫 innie.kext，是LILU.KEXT的插件，打开GITHUB去搜，并没有 innie的项目，再看远景，有一个帖子给了附件，是1.09的一个DEBUG版本

果断谷歌，最后在 macrumors上找到了这个名为 innie的LILU插件

也到找到其最新版本 1.2.0还是个RELEASE版本，这个也适用于白苹果

对于白苹果

  1. 关闭SLP

  2. 下载lilu.kext

  3. 打开LILU.KEXT包，在contents里建一个plugin，然后把 innie.kext扔进去

  4. 把lilu.kext扔进/Library/Extensions.

  5. 然后修复S/L/E权限，也就是：

    sudo chmod -R 755 /Library/Extensions/Lilu.kext
    sudo chown -R root:wheel /Library/Extensions/Lilu.kext
    sudo touch /Library/Extensions
    sudo kextcache -update-volume /

  6. 重启

对于黑苹果

  1. 挂载EFI分区，打开CLOVER/OC等的KEXT目录
  2. 打开LILU.KEXT包，在contents里建一个plugin，然后把 innie.kext扔进去
  3. 重启

效果

我自己的 雷克沙500G NVME SSD 在显示外置的时候，写1G/S 读1.3G/S，改完之后 写1.5G/S，读1.8G/S,速度还是有所提升的。
